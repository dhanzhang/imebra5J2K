Changes log¶

Version 5.2.1.1 changeset 906b3798 (this version)¶

  • Fixed bug #334 (The build of ImebraTests fails on some Windows machines) as per SOP SOP FIXBUG/2 Fixing a bug

Version 5.2.1.0¶

  • Fixed bug #329 (Slow SCP connection when running on Windows 10) as per SOP SOP FIXBUG/2 Fixing a bug

  • Fixed bug #330 (memoryImpl.h should include <string>)

  • Implemented enhancement #331 (Add example of C-FIND processing to scpexample) as per SOP SOP DEVENH/2 Implementing a new enhancement

Version 5.2.0.0¶

  • Fixed bug #323 (The PipeStream implementation throws StreamClosedError instead of returning 0 when the pipe has been closed) as per SOP SOP FIXBUG/2 Fixing a bug

  • Fixed bug #324 (The wrappers don’t have definitions for VOIDescription and VOIs) as per SOP SOP FIXBUG/2 Fixing a bug

  • Implemented enhancement #325 (Add an example that show the SCP feature) as per SOP SOP DEVENH/2 Implementing a new enhancement

  • Fixed bug #327 (The build fails on VS2015 because two dataHandlerNumeric functions look the same to the compiler, even if they are not) as per SOP SOP FIXBUG/2 Fixing a bug

  • Implemented enhancement #328 (Move the macos build from bitrise to appveyor) as per SOP SOP DEVENH/2 Implementing a new enhancement

Version 5.1.0.0¶

  • Implemented enhancement #316 (Add functions to retrieve floats, uint8, uint16, int8, int16) as per SOP SOP DEVENH/2 Implementing a new enhancement

  • Fixed bug #317 (readingDataHandlerStringAS::getAge does not properly check for the age units) as per SOP SOP FIXBUG/2 Fixing a bug

  • Implemented enhancement #318 (The data handler for VR “AT” should provide a way to retrieve the group and the id) as per sop SOP DEVENH/2 Implementing a new enhancement

  • Implemented enhancement #319 (Remove the compilation warnings on Windows) as per sop SOP DEVENH/2 Implementing a new enhancement

  • Fixed bug #320 (CMake tries to build the mm implementation files also when objective-c is not available) as per SOP SOP FIXBUG/2 Fixing a bug

  • Fixed bug #321 (The test objectivec.stringToNSStringTest fails) as per SOP SOP FIXBUG/2 Fixing a bug

Version 5.0.1.1¶

  • Fixed bug #307 (Remove the build-tools folder from the source code) as per SOP SOP FIXBUG/2 Fixing a bug

  • Implemented enhancement #308 (Remove non necessary files imebra_android.iml and ant.properties from the javaWrapper folder) as per SOP SOP DEVENH/2 Implementing a new enhancement

  • Implemented enhancement #309 (Add information about the commercial license in the user manual) as per SOP SOP DEVENH/2 Implementing a new enhancement

  • Implemented enhancement #310 (Migration from Mercurial to Git: change the SOPs & scripts to reflect the usage of Git instead of Mercurial) as per SOP SOP DEVENH/2 Implementing a new enhancement

  • Implemented enhancement #311 (Use the bitbucket pipeline artifacts instead of uploading the generated file to dropbox) as per SOP SOP DEVENH/2 Implementing a new enhancement

  • Implemented enhancement #312 (Add QtCreator .user files to gitignore) as per SOP SOP DEVENH/2 Implementing a new enhancement

  • Fixed bug #313 (Fix the CMake install commands so the Cmake configuration is properly exported and reusable by other modules) as per SOP SOP FIXBUG/2 Fixing a bug

  • Fixed bug #314 (The identifier for the Apple compiler changed from clang to AppleClang, causing the compilation to fail) as per SOP SOP FIXBUG/2 Fixing a bug

Version 5.0.1.0¶
